# Contributing to FortressJS

Thank you for your interest in contributing to FortressJS.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/davanesh/fortressjs.git
cd fortressjs
```

Install dependencies:

```bash
npm install
```

Build all packages:

```bash
npm run build
```

Run tests:

```bash
npm test
```

## Development Guidelines

* Follow the existing code style.
* Write clear and maintainable code.
* Add tests for new features.
* Update documentation when introducing new functionality.
* Keep pull requests focused on a single feature or fix.

## Reporting Issues

When opening an issue, include:

* FortressJS version
* Node.js version
* Steps to reproduce
* Expected behavior
* Actual behavior

## Pull Requests

1. Fork the repository.
2. Create a feature branch.
3. Commit changes with descriptive messages.
4. Ensure all tests pass.
5. Open a pull request.

Thank you for helping improve FortressJS.
